## Design Context

### Users
金融投资者、分析师、股票研究人员。使用场景：工作日盘中/盘后在桌面浏览器上查看 AI 生成的个股分析、市场概览和专家讨论。需要快速扫描大量数据，做出投资决策。注意力有限，信息密度高。

### Brand Personality
**精准、沉稳、专业** — 像一份高质量的卖方研报，而不是一个花哨的仪表盘。

### Aesthetic Direction
- **视觉基调**: Refined minimalism with editorial precision
- **主题**: Light mode (金融用户白天使用为主)
- **字体**: Satoshi (body, sans) + Erode (display headings, serif) — paired for warmth + authority
- **色彩**: Warm-indigo tinted neutrals (OKLCH hue 270), minimal accent usage (60-30-10 rule)
- **Anti-references**: 不要像 TradingView (过于密集), 不要像通用 AI dashboard (紫色渐变, 发光边框)

### Design Principles
1. **数据先行**: 数字和图表是主角，装饰性元素最少化
2. **层次分明**: 用字号对比 (≥1.25 ratio)、留白变化创建信息层级，不依赖卡片嵌套
3. **色彩克制**: 品牌色只用于 CTA 和关键状态指示，其余用 tinted neutrals
4. **动效有意义**: 只在状态转换 (加载、展开) 使用 ease-out-expo，禁止 bounce/elastic
5. **一致性**: 统一 4pt spacing scale, semantic shadow tokens, 禁止内联 arbitrary shadow

### Technical Tokens
- Fonts: `'Satoshi'` (body), `'Erode'` (headings)
- Spacing: 4pt scale (4, 8, 12, 16, 24, 32, 48, 64, 96)
- Shadows: `--shadow-xs` / `--shadow-sm` / `--shadow-md` / `--shadow-lg` / `--shadow-xl`
- Easing: `--ease-out` (expo), `--ease-in-out` (cubic)
- Colors: OKLCH-based, tinted toward hue 270 (warm indigo)
